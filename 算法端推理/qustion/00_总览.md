# 项目教学文档总览（src）
本目录用于帮助你快速理解 `/home/zyh/NewMedLabel/medlabel_image_cell_inference/src` 的整体工作方式、代码组织结构与端到端流转路径。

## 1. 项目一句话定位
这是一个“细胞推理”后台服务：从数据库领取 `pending` 任务 →（可选）通过 MinIO/本地预下载输入文件 → 生成瓦片坐标 → 多线程/多进程/GPU 流水线推理 → 异步写入数据库（并可选生成点图）→ 更新任务最终状态。

## 4. src 目录结构速览（按职责分块）
- `app/`：进程入口、CLI、应用生命周期、编排器工厂、信号与健康检查
- `config/`：基于环境变量的配置加载与强类型配置模型
- `db/`：asyncpg 连接池、Repository 层、LISTEN/NOTIFY 监听器与工厂注册表
- `task_core/`：通用任务框架（扫描/派发/执行/错误处理）
- `image_cell_processing/`：Image_cell 业务实现（扫描器、任务对象、任务管理器、编排器、推理流水线）
- `shared/`：跨模块共享能力（MinIO 客户端、统一存储服务、URL 解析、临时文件管理等）
